조인  
 **테이블이 n개가 있으면 n-1개의 조인이나와야함**
 **서브쿼리로 해결가능한건 조인쓰지말고 서브쿼리로 쓴다.**
 
하나이상의 테이블로부터 연관된 데이터를 검색해오는방법

FK = PK 조인(아닌경우도있음)


------------------

조인의종류


 inner join - 조인조건을 만족하는 행에대해서만 결과값이 나옴  
	     └	equijoin	: = 연산자조인 (이끼조인?)**
		join on	: on 명확하게 다적어줘야함			           - ANSI/ISO SQL 
		join using: 내츄럴조인이랑 비슷하지많 조건을 딱걸어줌                - ANSI/ISO SQL
		natural join : 조인조건이없다. =이름이 같은걸 알아서 조인걸어줌    - ANSI/ISO SQL
				(*필드가 같은게 하나일때걸어라 안그러면 쓰레기값나옴 )
 	 		
			          	
	
 outer join - 조인조건 만족안해도 출력이 가능해야하는 결과를 얻고자 할 때
	     회원 - 부서 


* 카티전조인 ex) 조인조건안하면 1000개 * 1000개 되서 백만개가 나옴 = 회사짤림 
	         		     (카티전 프로덕트)
------상세설명 ----

 equijoin: 가장 많이쓴다. 
	 where조건에 = 로 조인거는거임  (표준문법에 어긋나지 join 글자가없으니)
 	 테이블이 n개가 있으면 n-1개의 조인이나와야함
             ex : (처음두개 '조인'결과 테이블 + 다른테이블 '조인' )= 조인2개
	 (alias 사용해라)


 outter join - 회원 - 부서 가있을때 사장은 부서가없다. = 부서에 부서없음을넣을거냐?  NO
	     이럴때 아우터조인을쓴다.
